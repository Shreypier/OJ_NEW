Overview

                Codelang is a web-based platform that
                allows users to participate in programming
                contests, solve coding problems, and test 
                their algorithms against predefined test
                cases. 

                These platforms are popular among developers,
                students,and coding enthusiasts for improving their 
                problem-solving skills and competing in a 
                global community.
              






                
 Contest Format:


                1.Problems are usually of varying difficulty.
                2.Contests last for 2 hours.
                3.Participants are ranked based on the number
                  of problems they solve and the time taken.
                






                    
 Services:

    
1. Programming Contests:
Regular Contests: OJs host regularly scheduled contests where
participants compete to solve a set of problems within a fixed time.

2. Problem Sets:
Problem Library: A vast collection of problems categorized by difficulty, 
topic, and popularity.Problem Submission: Users can submit their solutions
to problems and receive immediate feedback on correctness and performance.

3. User Profiles and Ranking:
User Profiles: Track individual progress, submissions, and achievements.
Ranking Systems: Global and regional leaderboards based on performance in
contests and problem-solving.
